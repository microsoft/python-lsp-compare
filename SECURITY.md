# Security

## Reporting Security Issues

If you believe you have found a security vulnerability in this project, **do not**
open a public GitHub issue.

Instead, please report it responsibly via the
[Microsoft Security Response Center (MSRC)](https://www.microsoft.com/msrc/report-a-vulnerability).

MSRC will coordinate the investigation and response.

## Scope

This repository provides test cases, benchmarks, and tooling for comparing Python
Language Server Protocol (LSP) implementations. It is not intended to be deployed
as a production service.

However, security issues affecting users of this tooling (for example, issues that
could lead to unintended code execution when running benchmarks) should still be
reported via MSRC.
